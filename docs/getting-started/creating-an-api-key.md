# Creating an MPLAB ML API Key

<p align="center">
  <img src="images\mplab-ml-login.png" alt="MPLAB ML Login" width="100%">
</p>

1. Login to your MPLAB ML Development Suite account by visiting: https://www.mplabml.microchip.com/auth/login/

<p align="center">
  <img src="images\select-account-api-keys.png" alt="Select Account API Keys" width="100%">
</p>

2. In the upper right corner of the browser, click on your user account to see a menu, and select Account API Keys

<p align="center">
  <img src="images\generate-api-key.png" alt="Generate key" width="100%">
</p>

3. Click on the GENERATE API KEY button

<p align="center">
  <img src="images\named-key.png" alt="Provide a name for your key" width="100%">
</p>

4. Provide a memorable name for your key

<p align="center">
  <img src="images\save-key.png" alt="Save your key" width="100%">
</p>

5. Copy and save your key since you will not longer have access to it tthrough this menu

6. Use your new key when connecting from a Jupyter notebook.  The 01-authenication notebook (link to be added later) provides an example if this is your first time.
